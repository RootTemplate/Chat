-------- Упрошенное Использование --------

1. Запустите сервер. Один сервер - один чат. Запускайте сервер только ОДИН раз.
   Для запуска сервера выполните файл "bin/LaunchServer.bat".
   В открывшемся консольном окне будет выведен адрес сервера в формате "XXX.XXX.XXX.XXX:8079".
2. Для каждого пользователя чата запустите отдельный клиент.
   Для запуска клиента выполните файл "LaunchClient.bat".
   В открывшемся консольном окне 1) введите адрес сервера (см. 1 пункт); 2) свое имя в чате.
3. Для закрытия чата просто закройте окно сервера.



-------- Полное Использование --------

=== КЛИЕНТ:
start javaw -Dchat.printServerAddr=<true/false>
            -Dchat.consoleBufferSize=<int/"INF">
            -Dchat.fontSize=<int> -jar "ChatClient.jar" 
    <адрес сервера>
    <имя в чате>

1. <адрес сервера>. Адрес сервера в формате "хост:порт".
   Если порт не указан, то используется по умолчанию: 8079.
2. <имя в чате>. Ваше имя в чате.

1. "chat.printServerAddr" = <true/false>. Выводит или не выводит адрес сервера, к которому подключается.
   true - выводить; false - не выводить.
   По умолчанию "true".
2. "chat.consoleBufferSize" = <int/"INF">. Максимальное количество символом с консоли. Должно быть >= 0.
   Если указано "INF", то кол-во символом не ограничено.
   По умолчанию равен 25600 (символов).
3. "chat.fontSize" = <int>. Размер шрифта. Должен быть >= 1.
   По умолчанию равен 14.

Примичание. Чтобы запускать клиент не под Windows или с дополнительной консолью, замените "start javaw" в команде запуска на "java".


=== СЕРВЕР:
java -Dchat.maxNameLength=<int/"INF">
     -Dchat.maxMessageLength=<int/"INF">
     -Dchat.maxChatUsers=<int/"INF">
     -jar "ChatServer.jar" <порт сервера>

1. <порт сервера>. Порт запускаемого сервера или "DEF", тогда используется порт по умолчанию.
   Порт по умолчанию: 8079.

1. "chat.maxNameLength" = <int/"INF">. Максимальная длина имени пользователя в чате. Должна быть >= 0.
   Если указан "INF", то длина имени не ограничена.
   По умочланию равен 22.
2. "chat.maxMessageLength" = <int/"INF">. Максимальная длина сообщения в чате. Должна быть >= 0.
   Если указан "INF", то длина сообщения не ограничена.
   По умочланию равен 80.
3. "chat.maxChatUsers" = <int/"INF">. Максимальное количество пользователей в чате. Должно быть >= 0.
   Если указан "INF", то количество пользователей не ограничено.
   По умочланию равен 50.


Примичание. Все значения параметров, начинающихся с -Dchat, должны быть написаны в кавычках "".
